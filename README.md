# LIA-Backend-1

## 1: Node.js + Express

Built with Node.js, Express, and TypeScript using ES6 modules.

### Prerequisites

-   Node.js 18+
-   npm

### Quick Start guide:

1. Clone repo: https://github.com/Paajt/LIA-Backend-1.git
2. cd nodejs-express
3. npm install
4. npm run generate-token
5. Copy generated token
6. npm run dev
7. Go to http://localhost:3000/test.html
8. Try `Fetch Joke`-button (It should fail: Please enter a JWT token!)
9. Paste generated JWT token in `JWT Token field`
10. Press `Fetch Joke`-button.

-   Success!
